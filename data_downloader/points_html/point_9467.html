List of cookies used <p>Below is a list of the cookies we set on our websites and the purpose of each:</p> fastmail.com <ul> <li> <code>s_...</code> – Login cookie – this is used to allow you to log in to the Website.</li> <li> <code>f_...</code> – Security cookie – used as part of login and account recovery for keeping your account secure.</li> <li> <code>d_...</code> – Stores the domain you came in with, so we can keep you on the same domain when you log out.</li> <li> <code>lang_...</code> – Your preferred language to use for our interface once logged in.</li> <li> <code>theme_...</code> – Your preferred theme to use for our interface once logged in.</li> <li> <code>campaign</code> – If you came to our marketing site via a particular ad or campaign on a third party site, we set this so we can measure how effective the campaign was.</li> </ul> fastmailusercontent.com <ul> <li> <code>sk_…</code> Login cookie – this is used to authenticate your requests for message attachments and other user content.</li> </ul> topicbox.com <ul> <li> <code>s_...</code> – Login cookie – this is used to allow you to log in to the Website.</li> <li> <code>f_...</code> – Security cookie – used as part of login and account recovery for keeping your account secure.</li> </ul> pobox.com <ul> <li> <code>pobox_session</code> — Login cookie – this is used to allow you to log in to the Website.</li> <li> <code>browsertoken</code> — Security cookie – used as part of login and account recovery for keeping your account secure.</li> <li> <code>SpryMedia_DataTables_...</code> — Used for temporary storage of view parameters for data views (no user data)</li> </ul> listbox.com <ul> <li> <code>mason</code> — Login cookie – this is used to allow you to log in to the Website.</li
